# Lab 23: Style Switcher (jQuery)

## How to Run
1. Open the `23/` folder.
2. Open `index.html` in your browser.

## Requirements
- Browser.
- Internet (to load jQuery and Bootstrap CDNs).

## Example Input
- Click **Dark Mode**, **Blue Ocean**, or **Nature Green**.

## Expected Output
- The background color, text color, and form control styles change instantly across the entire page based on the selected theme.

## Notes
- Uses jQuery's `.css()` method to manipulate multiple elements at once.
- Demonstrates global style application without CSS files.
