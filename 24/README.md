# Lab 24: Age Calculator

## How to Run
1. Open the `24/` folder.
2. Open `index.html` in your browser.

## Requirements
- Browser.

## Example Input
- **Date of Birth:** `2000-01-01`

## Expected Output
- A clear message stating the age in **Years**, **Months**, and **Days** (relative to the current date).

## Notes
- The calculation accounts for the varying number of days in months and leap years implicitly via the JS `Date` object logic.
