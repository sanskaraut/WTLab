# Lab 16: Currency Converter (React)

## How to Run
1. Run this folder through a local server (like XAMPP or VS Code **Live Server**).
2. Open `http://localhost/WTLab/16/index.html`.

## Requirements
- Browser.
- Internet (React CDN).

## Example Input
- **Amount in USD:** `10`

## Expected Output
- The display immediately updates to show **₹ 835.00** (assuming a rate of 83.50).
- The conversion happens in real-time as you type.

## Notes
- Uses `useState` for bi-directional data binding.
- Uses `toLocaleString` to format the rupee amount in Indian currency format.
