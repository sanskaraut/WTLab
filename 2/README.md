# Lab 2: Tabs Website (SDP, EDI, DT, Course Projects)

## How to Run
1. Open the `2/` folder.
2. Open `index.html` in your web browser.

## Requirements
- Web Browser.
- Internet (for Bootstrap CDN).

## Example Input
- Click on the tabs: **SDP**, **EDI**, **DT**, or **Course Projects**.

## Expected Output
- The content below the tabs should change dynamically without reloading the page.
- Each tab displays specific information about the selected VIT curriculum component.

## Notes
- Uses Bootstrap's `nav-pills` and `tab-content` for the dynamic switching.
