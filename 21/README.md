# Lab 21: Student Records Edit & Delete

## How to Run
1. Start XAMPP/WAMP (Apache + MySQL).
2. Place the `21/` folder in `htdocs`.
3. Open `http://localhost/WTLab/21/index.php`.

## Requirements
- MySQL (database `student_db`).

## Example Input
- **Edit:** Change the "Name" or "Email" of an existing student in the table.
- **Update:** Click the **Update** button next to that row.
- **Delete:** Click the **Delete** button and confirm the browser alert.

## Expected Output
- The table updates immediately to show the modified data.
- Deleted rows disappear from the list.

## Notes
- This lab focuses on the **Update** and **Delete** aspects of CRUD.
- Uses inline editing within the table for a cleaner demo experience.
