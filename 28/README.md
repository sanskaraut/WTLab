# Lab 28: Light & Dark Mode Toggle

## How to Run
1. Open the `28/` folder.
2. Open `index.html` in your browser.

## Requirements
- Browser.
- Internet (React CDN).

## Example Input
- Click the **Switch to Dark Mode** button.

## Expected Output
- The background color of the entire screen changes to dark grey/black.
- The text color changes to white.
- The button label and theme status update dynamically.

## Notes
- Uses CSS transitions for a smooth color change effect.
- The theme state is global within the App component.
