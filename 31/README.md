# Lab 31: System Notifications (Redux)

## How to Run
1. Run this folder through a local server (like XAMPP or VS Code **Live Server**).
2. Open `http://localhost/WTLab/31/index.html`.

## Requirements
- Browser.
- Internet (React & Redux CDNs).

## Example Input
- Click the **Add Success**, **Add Error**, or **Add Warning** buttons.

## Expected Output
- A toast notification slides in from the right.
- Multiple notifications can be stacked.
- Notifications disappear when the "X" button is clicked OR automatically after 5 seconds.

## Notes
- Demonstrates how to handle a dynamic list of objects in Redux.
- Shows how to combine Redux `dispatch` with standard JavaScript `setTimeout` for time-based state changes.
