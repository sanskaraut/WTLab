# Lab 1: CV with Video Background

## How to Run
1. Open the `1/` folder.
2. Double-click `index.html` to open it in your default web browser.

## Requirements
- Modern Web Browser (Chrome, Firefox, Edge).
- Internet connection (to load Bootstrap CSS and the background video).

## Example Input
- No input required; the page displays static CV data.

## Expected Output
- A visually appealing, responsive CV page with an animated video background.
- Multiple columns layout (Skills/Languages on left, Education/Experience on right).

## Notes
- The video is loaded from a CDN. If the video doesn't load, check your internet connection.
