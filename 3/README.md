# Lab 3: JavaScript Array Operations

## How to Run
1. Open the `3/` folder.
2. Open `index.html` in your browser.

## Requirements
- Any modern web browser.

## Example Input
- **Array:** `40, 10, 30, 20, 50`
- **Search Element:** `30`

## Expected Output
- **Reverse:** `[ 50, 20, 30, 10, 40 ]`
- **Sort (Asc):** `[ 10, 20, 30, 40, 50 ]`
- **Search Result:** `Element '30' found at index 2` (assuming original array)

## Notes
- The program handles both string and numeric data, though sorting is optimized for numbers.
- Uses built-in JS methods like `.reverse()`, `.sort()`, and `.indexOf()`.
