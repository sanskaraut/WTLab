# Lab 25: Traffic Signal Lights

## How to Run
1. Open the `25/` folder.
2. Open `index.html` in your browser.

## Requirements
- Browser.

## Example Input
- Click the **Red**, **Yellow**, or **Green** buttons to change manually.
- Click **Auto Mode** to start an automatic loop.

## Expected Output
- The corresponding circle in the traffic light box glows with a bright LED-like shadow effect.
- The status text below changes to "STOP", "READY", or "GO" with appropriate colors.

## Notes
- Uses CSS `box-shadow` to simulate the glowing LED effect.
- Manual mode and Auto mode (using `setInterval`) are both implemented.
