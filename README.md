# Web Technology Lab Repository

This repository contains 35 lab problems implemented across various tech stacks, optimized for lab exam demonstrations.

## Repository Structure

- `1/` to `35/`: Individual lab folders.
- Each folder contains:
  - Source Code (HTML, PHP, Node.js, React, or Spring Boot)
  - `README.md`: Step-by-step run instructions and demo notes.
  - `explanation.txt`: Conceptual overview for viva/explanation.

## Technology Mapping

| Category | Lab Numbers | Requirements |
|----------|-------------|--------------|
| **HTML + JS** | 1, 2, 3, 22, 23, 24, 25 | Browser |
| **PHP + MySQL** | 4, 5, 6, 11, 12, 13, 15, 17, 18, 19, 20, 21 | XAMPP / WAMP |
| **React (CDN)** | 7, 8, 16, 28, 29, 30, 31 | Browser (Internet for CDNs) |
| **Node.js** | 10, 27, 34, 35 | Node.js installed |
| **Spring Boot** | 9, 14, 26, 32, 33 | JDK 11+, IDE (IntelliJ/Eclipse) |

## Quick Demo Guide

1.  **HTML/React Labs:** Just open `index.html` in any modern browser.
2.  **PHP Labs:** Move the folder to `htdocs` (XAMPP) and access via `localhost`. SQL setup is in the `README.md`.
3.  **Node.js Labs:** Run `npm install` and `node app.js`.
4.  **Spring Boot Labs:** Open as a Maven project in your IDE and run the main class.
