# Lab 17: Waste Collection System

## How to Run
1. Place the `17/` folder in `htdocs`.
2. Open `http://localhost/WTLab/17/index.php`.

## Requirements
- PHP environment.

## Example Input
- **Type:** `Plastic`
- **Location:** `Canteen Area`
- **Phone:** `9898989898`

## Expected Output
- A confirmation message stating that the request is registered.
- Dynamically identifies the correct authority (e.g., "Plastic Recycling Cell") based on the waste type.
- Generates a random Ticket ID for tracking.

## Notes
- Demonstrates conditional logic to route requests to different "departments".
