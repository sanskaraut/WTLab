# Lab 22: Canvas Tag Drawing

## How to Run
1. Open the `22/` folder.
2. Open `index.html` in your browser.

## Requirements
- Browser with HTML5 Canvas support (almost all modern browsers).

## Example Input
- Select **Draw Rectangle** or **Draw Line**.
- Click on the canvas, drag the mouse, and release.

## Expected Output
- A rectangle (blue) or a line (red) appears on the canvas based on your mouse movement.

## Notes
- `mousedown` captures the start coordinates.
- `mouseup` captures the end coordinates and triggers the drawing function.
